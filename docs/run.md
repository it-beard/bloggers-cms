## Commands for running code locally
#### Move to the project folder
`cd Pds`

#### In Watch mode
`dotnet watch -v run --project Pds.Web`

`dotnet watch -v run --project Pds.Api` 

#### In Release mode
`dotnet run --project Pds.Web`

`dotnet run --project Pds.Api` 

#### Additional commands

`dotnet clean`

`dotnet build`
